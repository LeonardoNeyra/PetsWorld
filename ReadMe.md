## **Proyecto de DAM**
>Integrantes:  
Corcuera Enco, Alem :pig:  
Jara Carpio, Carlos :bird:  
Neyra Ocaña, Leonardo :penguin:  
El proyecto se llama "PetsWorld :paw_prints:" y pretende ser una red social donde personas pueden crear post para dar en adopción a una mascota; de la misma manera otros usuarios pueden ver el post y ponerse en contacto con la persona que creó el post para adoptar a la mascota del post.
